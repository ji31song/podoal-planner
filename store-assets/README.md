# Google Play 이미지

- `icon-source.png`: 사용자가 선택한 원본 캐릭터 이미지
- `store-icon.png`: 512×512 PNG, Play Console 앱 아이콘
- `feature-graphic.png`: 1024×500 PNG, Play Console 그래픽 이미지
- `screenshots/`: 실제 앱 화면에서 개인정보를 가린 휴대전화 스크린샷

아이콘과 시작 화면 이미지는 `scripts/generate-brand-assets.cjs`로 같은 원본에서 생성합니다.
스크린샷에는 실제 가족 이름, 이메일, 전화번호와 초대 코드를 사용하지 않습니다.
